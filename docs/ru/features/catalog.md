# Каталог товаров и Склад 📦

Этот модуль управляет всем, что связано с товарами, категориями и остатками на складе.

## 🗄️ Модель данных
- **Product**: Основная модель (Название, крепость ABV, горечь IBU, цена, остаток).
- **Category**: Иерархическая структура категорий товаров.
- **Stock**: Отслеживание остатков в реальном времени.

## ⚙️ Технические особенности
- **Импорт каталога**: Скрипт `features/products/services/import_catalog.py` отвечает за массовое обновление данных из внешних источников.
- **Selectors**: Используются для фильтрации товаров по категориям, рейтингу и наличию.

## 📦 Валидация остатков
Наличие товара строго проверяется при оформлении заказа и при добавлении в корзину. Мы гарантируем, что пользователь не сможет заказать больше товара, чем есть на складе.

## 🎨 Интерфейс (UI)
- **Фильтрация**: Списки товаров обновляются через HTMX для мгновенного отклика.
- **Поиск**: Полнотекстовый поиск по всей базе товаров.
